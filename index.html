No meu portfólio você encontrará projetos desenvolvidos durante imersões, outros realizados em sala de aula, e alguns criados por mim mesmo.

1. Carrossel de Imagens

Carrossel de imagens insprado no desenho das Winx, feito com a estrutura de caixas onde uma contem as imagens para ser fixada de fundo, outra informações e um para os botões, utilizando como complemento para aprimorar os arquivos .css e .js, com o css usei classes para esconder as imagens e as informações e apenas mostrar as que estão com a classe selecionandas, e este projeto conta com responsividade utilizando CSS, e com o js usei a forEach e adicionei o evento para que cada botão troque as de acordo com o índice de cada botão clicado.

2. Cards

Cards inspirado no desenho do Pokemon, feito com a estrutura de listas não ordenadas, cada item da lista é uma carta, com a imagem, título e descrição, utilizando o css para aprimorar o design e deixá-lo responsivo, e o js para a função de clique, quando o usuário clicar no botão da direita ou esquerda, ira mudar a carta para próxima ou anterior na sequência da lista.

3. Formulário Interativo

Formulário Interativo, com a ajuda de uma API do SheetDB no atributo action e o method post, feito por etapas com JavaScript, todas as respostas preenchidas serão colocados em uma planilha de minha posse, assim que o botão de enviar for clicado, ao final aparecerá uma mensagem escrita "entraremos em contato em breve" não recarregando ou redirecionando para outras paginas, feito com inputs e labels em caixas, utilizando o css para aprimorar o design, e JavaScript para a interatividade.

4. Seletor de Cards

Seletor de Cards é inspirado no desenho Hora de Aventura e foi desenvolvido durante um dos imersivos do canal Dev em Dobro. Utilizei uma estrutura de caixas para cada carta dos diversos personagens da animação e uma estrutura de lista para que o usuário selecione um personagem a ser exibido. Toda a aparência e a responsividade deste projeto foram feitas com CSS, enquanto a interatividade para troca das cartas selecionadas foi implementada com JavaScript, utilizando o método forEach para garantir que cada item da lista selecionado exiba a carta correta.

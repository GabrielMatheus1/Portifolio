Meu Portifólio

Carrossel de Imagens

Carrossel de imagens insprado no desenho das Winx, feito com a estrutura de caixas onde uma contem as imagens para ser fixada de fundo, outra informações e um para os botões, utilizando como complemento para aprimorar os arquivos .css e .js, com o css usei classes para esconder as imagens e as informações e apenas mostrar as que estão com a classe selecionandas, e com o js usei a forEach e adicionei o evento para que cada botão troque as de acordo com o índice de cada botão clicado.

Cards

Cards inspirado no desenho do Pokemon, feito com a estrutura de listas não ordenadas, cada item da lista é uma carta, com a imagem, título e descrição, utilizando o css para aprimorar o design, e o js para a função de clique, quando o usuário clicar no botão da direita ou esquerda, ira mudar a carta para próxima ou anterior na sequência da lista.

Formulário Interativo

Formulário Interativo, com a ajuda de uma API do SheetDB no atributo action e o method post, feito por etapas com JavaScript, todas as respostas preenchidas serão colocados em uma planilha de minha posse, assim que o botão de enviar for clicado, ao final aparecerá uma mensagem escrita "entraremos em contato em breve" não recarregando ou redirecionando para outras paginas , feito com inputs e labels em caixas, utilizando o css para aprimorar o design, e JavaScript para a interatividade.

Seletor de Cards

Seletor de Cards inspirado no desenho Hora de Aventura, selecionei algumas imagens de personagens para este projeto e cada um deles tem sua descrição, na lista ao lado do card tem outros personagens sendo possível substituir o card visível atual, alterei a imagem do cursor do mouse para uma Espadinha, neste projeto utilizei HTML para estrutura, CSS para estilização da pagina, e JavaScript para a interação da pagina como a troca de personagens.

HFA - Landing Page

Landing Page feito para a empresa HFA School, uma pagina de captura para novos usuarios, este projeto foi desenvolvido por 3 pessoas, Jonathan (Prototipação do projeto), eu (sessões dos Professores, Formulario Interativo, FAQ e Redes Sociais) e Ketlhyn (sessão do video que prendera o usuario até que ele acabe e Rodapé da pagina). Utilizando HTML, CSS, JavaScript e APIs.
